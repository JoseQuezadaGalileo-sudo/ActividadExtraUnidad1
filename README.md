# ActividadExtraUnidad1
Contiene los archivos y carpetas solicitados para la tarea extra de la unidad 1
El motivo de haber elegido la licencia Apache 2.0 se debe a que protege a los desarrolladores de las patentes de problemas legales relacionadas con las mismas, y no obliga a compartir mejoras hechas.
Además es de uso comercial y permite su aokicación sin tener problemas relacionados con las patentes de código fuente.
